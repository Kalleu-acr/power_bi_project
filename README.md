# Projeto BI: Censo Escolar 2023 e SAEB 2023

## Descrição do Projeto
A educação básica configura-se como fundamento essencial para o desenvolvimento social, econômico e cultural. Este projeto acadêmico de Business Intelligence (BI) tem como objetivo investigar como características estruturais das escolas — mensuradas pelo Censo Escolar — se relacionam com os resultados observados nas avaliações do SAEB.

O estudo integra, organiza e explora grandes volumes de informações públicas, permitindo identificar padrões, correlações e sinais relevantes que podem subsidiar decisões de políticas públicas e gestão escolar.

## Motivação
Apesar da riqueza de dados disponíveis no Censo Escolar e no SAEB, ainda há lacunas sobre como características concretas do ambiente escolar se correlacionam com os resultados de aprendizagem. Aspectos como laboratórios de informática, acesso à internet, quadras esportivas e instalações adaptadas têm impacto potencialmente significativo no desempenho estudantil.

## Fontes de Dados
- **Censo Escolar 2023 (INEP):** dados sobre escolas, turmas, matrículas, infraestrutura e pessoal.  
- **SAEB 2023 (INEP):** dados de avaliações em larga escala, indicadores de aprendizagem e questionários de estudantes, professores e gestores.

## Objetivo
Analisar a relação entre a infraestrutura das escolas e os resultados educacionais, utilizando técnicas de BI e análise de dados para:  
1. Integrar e organizar dados de diferentes bases.  
2. Gerar insights sobre padrões e correlações entre capital físico das instituições e desempenho escolar.  
3. Subsidiar decisões de política e gestão educacional.
